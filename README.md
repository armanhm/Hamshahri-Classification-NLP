# Hamshahri-Classification-NLP
 Hamshahri dataset Classification using National Language Proccessing
 
 The technologies and frameworks used in this project are:
 * hazm
 * numpy
 * nltk
 * pandas
 * sklearn
 
 The Final accuracy in kaggle report was: %78.39
